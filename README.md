Ce Git a été créé dans le cadre d'un entretien technique pour un poste d'ingénieur d'étude en traitement d'images et de signaux.

Le contexte ici est celui de la modélisation des interactions dans un bloc opératoire.

Ce git contient:
  - un rapport pour la partie data / préparation au format pdf
  - un Jupyter Notebook pour la partie data / synchronisation
  - le fichier csv contenant les données liées à la fréquence cardiaque
